# Plant Disease Detection using YOLO

This project demonstrates object detection using a YOLO-based workflow for identifying diseased plant leaves.

## Features
- Plant disease detection
- Bounding box predictions
- Computer vision workflow
- AI-assisted object detection

## Tools Used
- Roboflow
- YOLO
- Computer Vision
- GitHub

## Output
The model detects diseased regions on plant leaves using bounding boxes.
