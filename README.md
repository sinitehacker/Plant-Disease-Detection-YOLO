# Plant Disease Detection using YOLO

This project demonstrates a YOLO-based object detection workflow for identifying diseased plant leaves using computer vision and AI tools.

## Project Overview

The workflow detects diseased regions in plant leaves by generating bounding boxes around infected areas. This project was created as part of an AI internship focusing on computer vision and object detection.

## Features

* Plant disease detection
* Object detection using YOLO
* Bounding box predictions
* AI-assisted workflow
* Computer vision implementation

## Tools & Technologies

* Roboflow
* YOLO
* GitHub
* Computer Vision
* Artificial Intelligence

## Output

The model successfully detects diseased portions of leaves and highlights them with bounding boxes and prediction outputs.
